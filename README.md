project - Personal Finance Tracker

Project Overview -
A Python-based program written to track our daily expenses and understand our spending habits. This project is inspired by the "Finance & Banking" category 

Features -
1. Add Expenses: add daily expenses with category and description.
2. Data Persistence: saves data to a CSV file so records are not lost when we exit the code.
3. View History: sows you a table typee report of all past expenses and shows the total expenses.
4. Visualization: you can Generate a Pie Chart using Matplotlib to visualize your expenditure.

Requirements -
1. Python 3.x
2. Pandas Library
3. Matplotlib Library

How to Run -
1.copy the code file
2. install requirements
